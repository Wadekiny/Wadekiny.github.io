设置系统代理

如果只是启动了软件的代理，系统的流量并没有从代理的端口走，还是不能git加速。

  • 对于终端，要设置环境变量才能走代理export all_proxy=socks5://127.0.0.1:7890
  • 对于gnome桌面，有系统的手动代理管理
  • 对于git，在~/.gitconfig中设置：
# linux # network # proxy # git
