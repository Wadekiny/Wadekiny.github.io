设备

使用多设备时，张量之间的操作必须存在于同一个设备

Rank（阶数）

常量： rank 0
向量： rank 1
矩阵： rank 2
n维张量： rank n

Shape（形状）

指明每一层有多少个元素

使用现有的数据创建张量

  1. 构造函数，使用默认的数据类型，复制数据

  2. （推荐）工厂函数，使用相同的数据类型（或人为指定），复制数据
